## A full-stack web application with <img src="https://raw.githubusercontent.com/danielcranney/readme-generator/main/public/icons/skills/svelte-colored.svg" width="24" height="24" alt="Svelte" style="vertical-align: middle;" /> frontend and <img src="https://raw.githubusercontent.com/danielcranney/readme-generator/main/public/icons/skills/go-colored.svg" width="32" height="32" alt="Golang" style="vertical-align: middle;" /> backend

### Prerequisites

- [Bun](https://bun.sh) and [Golang](https://go.dev) installed

### Install dependencies

```sh
bun install
```

### Development

> Open [localhost:5173](http://localhost:5173) in your browser and use:

```sh
bun run dev
```

### Build

```sh
bun run build
```

> Preview the production build:

```sh
bun run preview
```

### Use conventional commits

```sh
feat: add SvelteKit frontend scaffold and project configuration
```
